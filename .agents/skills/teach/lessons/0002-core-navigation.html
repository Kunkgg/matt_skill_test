Yazi 课程 · 第 1 课

核心导航：hjkl、gg、G、z、Z

好消息：八成的键和 ranger / lf / vim 一模一样。重点是 yazi 独有的两个跳转键。

本课的胜利

纯键盘在目录树里上下穿梭，进出自如；并且能用 Z 一键瞬移到你常用的项目目录。

四个移动键（和 vim / ranger / lf 完全一致）

键动作
h回到上一级父目录
l进入光标所在的子目录
j光标下移一项
k光标上移一项
肌肉记忆迁移

你 ranger / lf 里这套手感直接照搬就行。方向键 ←↓↑→ 也同样可用，但既然你熟 vim，就别退回方向键了。

列表内大跳

键动作
g g跳到列表最顶端（先 g 再 g）
G跳到列表最底端

大写 J / K：滚的是预览，不是列表

键动作
K在右侧预览窗里向上滚动 5 行
J在右侧预览窗里向下滚动 5 行
易混点

小写 jk 移动的是文件列表里的光标；大写 JK 滚动的是预览内容。当你预览一个长文件、想看下面的内容时，就用大写 J。这是 yazi 一个顺手的小设计，ranger/lf 里没有对应物。

两个跳转键（yazi 的杀手锏）

这是「快速跳转目录」的核心。普通移动靠 hl，但跨越大目录树就得靠这两个：

键动作底层
z模糊跳转：输几个字母匹配目录fzf（按当前目录扫描）
Z常用度跳转：瞬移到你最常去的目录zoxide（按访问频率）
g Space交互式输入路径跳转—
刚才发生了什么

我发现你的 zoxide 装了却没接进 shell（db 是空的），所以 Z 本来是死的。我已经做了两件事：① 在 .zshrc 加了 eval "$(zoxide init zsh)"，于是 shell 里有了 z 命令；② 往 zoxide 喂了 ~、~/Repos、~/Repos/matt_skill_test 三个真实目录，让 Z 现在就能用。

让它越用越聪明：以后在 shell 里多用 z <目录片段> 代替 cd，zoxide 会记住你常去的地方，Z 就会把最常用的顶到最前面。

动手验证

在你的机器上跑一遍
  1. exec zsh（让 zoxide 和 y 都生效）。
  2. cd ~ && y 从家目录启动。
  3. l 进入 Repos，再 l 进入 matt_skill_test；然后 h h 退回 家目录。
  4. gg 到顶，G 到底，感受大跳。
  5. 按 z，输入 matt，回车 —— 模糊跳进 matt_skill_test。
  6. 按 Z，看到 zoxide 列出的目录，选一个回车 —— 瞬移过去。
  7. q 退出，pwd 应显示你最后所在的目录（第 0 课的 y 在收尾）。
检索练习 · 先想再点开

1. 用 fzf 模糊跳转目录，按哪个键？

2. 用 zoxide 按常用度跳转，按哪个键？

3. 进入光标所在的子目录，按哪个键？

4. 回到上一级父目录，按哪个键？

看答案

1. z　　2. Z　　3. l　　4. h

记忆钩子：小写看当前目录（fzf 扫描当下），大写看全局历史（zoxide 记住你常去的）。
